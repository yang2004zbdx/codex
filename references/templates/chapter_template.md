# 章节写作模板

## 结构要求
- 每个一级或二级标题下至少包含一个主题句。
- 每个主题句后应有 2-4 个支撑句。
- 每个主题块后应有 1 个过渡句或小结句。

## 段落模板
```
主题句：本段/本小节的核心论点。

支撑句 1：已有研究或实验证据。
支撑句 2：与本节主题相关的结论或比较。
支撑句 3：如需要，可补充局限性或争议。
分析句：该证据对本综述论点意味着什么。
过渡句：自然引出下一节或下一段。
```

## 章节推进模板
- 引言：背景 → 问题 → 综述范围 → 本章安排
- 研究进展：按主题/方法分组 → 核心结论 → 局限或争议
- 问题与展望：已有进展 → 关键缺口 → 未来方向
- 结论：总结核心发现 → 概括主要贡献 → 指出仍需解决的问题

## 写作禁忌
- 不要连续两段都以“某作者报道了……”开头。
- 不要只罗列文献，不做比较分析。
- 不要每段都表达完全相同语气。
- 不要用太多“近年来，……引起广泛关注”这类空泛表达。
