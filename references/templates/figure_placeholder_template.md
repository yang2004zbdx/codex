# 图片占位模板

## 用途
当图片无法自动提取、生成或排版时，插入结构化占位块，避免缺失图片导致结构断裂。

## 占位块格式
```
[图片占位]
- 建议图名：
- 建议位置：章节 / 段落
- 图表目的：
- 推荐引用来源：作者（年份）《标题》，出版物，Figure/Table 编号，页码
- 版权状态：可引用 / 需授权 / 需改绘
- 可执行动作：复制原图 / 改绘 / 联系版权方 / 重做示意图
```

## 使用规则
- 每个占位块必须包含上述全部信息。
- 在 `<OUTPUT_DOC>` 中保留占位块，不得直接删除。
- 占位块需与正文论点严格对应。

## 占位提示
- `<TOPIC>`
- `<FIGURE_FOLDER>`
- `<CORE_LITERATURE>`
- `<BENCHMARK_PAPERS>`
