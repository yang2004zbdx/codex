# 25 期刊选择与 Cover Letter

## 目标
根据正文内容和目标期刊要求完成投前准备。

## 输入
- 终稿文档
- `<TARGET_JOURNAL>`
- `<REFERENCE_STYLE>`

## 执行动作
1. 评估期刊范围与主题匹配度。
2. 阅读目标期刊作者指南。
3. 撰写 Cover Letter：
   - 说明综述创新点
   - 说明对期刊读者的价值
   - 声明无利益冲突、未一稿多投
4. 按要求调整格式。

## 交付物
- Cover Letter
- 投稿准备说明

## 检查点
- Cover Letter 是否说明综述贡献？
- 是否已匹配目标期刊格式？
