# 02 导入 Zotero 并整理

## 目标
把已有文献统一导入 Zotero，形成可检索、可分组、可导出题录的文献库。

## 输入
- `<LITERATURE_FOLDER>`
- `<REFERENCE_STYLE>`

## 执行动作
1. 检查 `<LITERATURE_FOLDER>` 中的题录来源：
   - PDF 内置元数据
   - RIS / BibTeX / CSV
   - Word 题录清单
2. 导入 Zotero：
   - 按来源批次导入
   - 保留原始文件路径或复制到 Zotero 存储目录
3. 补全字段：
   - 标题、作者、期刊、年份、DOI、摘要、页码
4. 建立分类：
   - 按文献类型分组：综述、研究、方法、背景
   - 按主题标签分组：与 `<TOPIC>` 相关的子主题
5. 建立笔记字段：
   - 每篇至少记录：研究问题、方法、关键结果、可引用点、与综述关系
6. 导出：
   - 导出为写作插件可用格式
   - 导出为投稿格式

## 交付物
- Zotero 文献库说明
- 分组与标签规则
- 题录导出说明

## 检查点
- 是否能按主题快速检索到某篇文献？
- 是否所有核心文献都有完整题录？
- 是否已区分参考综述和研究文献？
