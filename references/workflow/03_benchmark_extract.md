# 03 参考综述格式抽取

## 目标
从 `<BENCHMARK_PAPERS>` 中抽取结构、表达、图表风格，形成写作参考。

## 输入
- `<BENCHMARK_PAPERS>`
- `<OUTPUT_DOC>`

## 执行动作
1. 精读 1-3 篇参考综述。
2. 抽取结构：
   - 一级标题设置逻辑
   - 二级、三级标题组织方式
   - 摘要、引言、结论写法
3. 抽取表达：
   - 常用句式
   - 学术语气与转述方式
4. 抽取图表：
   - 图名、表名写法
   - 图表位置与编号规则
5. 形成格式说明文档，供后续章节模仿。

## 交付物
- 格式抽取说明
- 可复用句式清单

## 检查点
- 是否已明确参考综述的目录结构？
- 是否已提炼 5-10 个可复用句式？
- 是否已说明图表标题规范？
