# 14 图表规划

## 目标
规划全文图表，明确哪些图必须原创、哪些可引用、哪些需要 fallback 占位。

## 输入
- 细化目录
- `<FIGURE_FOLDER>`
- `<FIGURES_TABLES_FILE>`
- `<CORE_LITERATURE>`
- `<BENCHMARK_PAPERS>`

## 执行动作
1. 盘点图表类型：
   - 体系图、流程图、结构示意图
   - 对比表、性能表
2. 规划图表位置与作用。
3. 制定图表标题规范。
4. 判断版权需求：
   - 可引用并标注来源
   - 需申请版权
   - 需改绘
5. 若无法自动生成图片，必须插入图片占位块。

## 交付物
- 图表清单
- 图表占位说明

## 检查点
- 每张图是否都能回答“它为什么必须出现在综述中”？
- 是否已说明图片来源与版权状态？
