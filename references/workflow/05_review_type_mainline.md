# 05 综述类型与核心主线确定

## 目标
确定综述类型与全文梳理主线，避免交叉混乱。

## 输入
- 选题结果
- 文献盘点结果

## 执行动作
1. 选定综述类型：
   - 叙述性综述（主流）
   - 系统性综述
   - Meta 分析综述
2. 确定核心主线，选择一种：
   - 按时间发展
   - 按方法分类
   - 按机理差异
   - 按应用场景
3. 说明为什么该主线最适合 `<TOPIC>`。
4. 明确全文只使用一条主线。

## 交付物
- 综述类型说明
- 核心主线说明

## 检查点
- 是否只选定一种综述类型？
- 是否明确全文只用一条主线？
- 是否说明主线与选题的匹配理由？
