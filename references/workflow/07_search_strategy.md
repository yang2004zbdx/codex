# 07 检索策略设计

## 目标
设计多数据库组合检索式，确保检索结果覆盖核心文献。

## 输入
- `<TOPIC>`
- `<DATABASE_LIST>`
- `<LITERATURE_FOLDER>`

## 执行动作
1. 列出计划检索的数据库。
2. 设计检索式：
   - 主题词 + 同义词
   - 布尔逻辑组合
   - 排除无关领域
3. 设置筛选条件：
   - 时间范围（建议近 5 年为主）
   - 文献类型（研究、综述、方法）
   - 语言（中英文）
4. 记录检索策略，便于复现。

## 交付物
- 检索策略文档
- 检索式清单

## 检查点
- 检索式是否能精确锁定 `<TOPIC>`？
- 是否已记录数据库与筛选条件？
