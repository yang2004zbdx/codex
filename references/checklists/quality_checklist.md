# 质量检查清单

## 使用说明
用于检查综述写作质量是否达到可投稿水平。

## 检查项
- [ ] 选题是否聚焦，3 个关键词可锁定范围？
- [ ] 是否明确综述类型与核心主线？
- [ ] 摘要是否涵盖背景、分类、痛点、展望？
- [ ] 主体章节是否按主线组织，没有交叉混乱？
- [ ] 是否每 3-5 篇文献后有对比评述？
- [ ] 问题与不足是否系统总结？
- [ ] 展望是否对应痛点？
- [ ] 图表是否服务论点而非装饰？
- [ ] 图片占位块是否写清来源与版权说明？
- [ ] Zotero 是否已按主题分组并补全字段？
- [ ] 参考文献格式是否统一？
- [ ] 是否存在大段照搬原文？
- [ ] 专业名词是否全文统一？
- [ ] 目录、图表、引文、参考文献是否一致？
