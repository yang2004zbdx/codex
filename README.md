---

**review-writing-sci** 是一个面向 SCI/中文核心综述写作的通用 Skill 模板。  
它把完整的综述写作过程拆成 **25 个标准化步骤**，从已有文献盘点、Zotero 导入整理、参考综述格式抽取、选题收敛、文献筛选精读，到框架搭建、图表规划、正文撰写、逻辑自查、降重润色，再到期刊匹配与 Cover Letter，形成了一条可重复执行的写作流水线。

这个 Skill 的核心特点是**主题无关、模板化、可复用**：
- 不绑定任何具体课题、作者或期刊
- 所有正文提示、模板和检查清单都使用占位符，换一个题目就能直接用
- 特别强化了 Zotero 工作流：导入、分组、标签、笔记、终检、导出格式
- 特别强化了 图片处理机制：优先从文献中选取可引用图片；若无法自动提取或生成，则强制使用图片占位块，写清建议图名、建议位置、推荐引用来源、版权状态和下一步动作

如果你已有若干 PDF、RIS、BibTeX 或 Word 题录，可以直接套用这套 Skill，按 Step 1 → Step 25 的顺序推进；如果你只有初步想法，也可以从 Step 4 选题收敛开始。整个 Skill 既适合个人写综述，也适合团队沉淀成标准化写作模板。
